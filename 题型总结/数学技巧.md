#### 338.比特位计数

给定一个非负整数 num。对于 0 ≤ i ≤ num 范围中的每个数字 i ，计算其二进制数中的 1 的数目并将它们作为数组返回。

```
示例1：
输入: 2
输出: [0,1,1]
示例2：
输入: 5
输出: [0,1,1,2,1,2]
```

###### 题解：

````c++
/***
奇数的1的个数为比它小1的数字拥有的1的个数，而偶数个数和÷2的数字1的个数相同e.g:4的二进制为100，和2的二进制10的1个数相同。
***/
class Solution {
public:
    vector<int> countBits(int num) {
        vector<int> res(num+1);
        res[0]=0;
        for(int i=1;i<=num;i++){
            if(i&1)//i&1==1则为奇数 异或运算
                res[i]=res[i-1]+1;
            else res[i]=res[i/2];
        }
        return res;
    }
};
```
